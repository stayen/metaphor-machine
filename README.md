# Metaphor Machine Suite

Metaphor Machine suite: a development project to implement a set of programmatic tools/reference of AI assisted techniques to assist creation of generative music tracks in Suno/Producer.ai services.

Currently implemented: **Metaphor Machine** Python script. Read the corresponding README.md in its directory

Last update: now available as Python module, i.e.

```
pip install metaphor-machine
metaphor
```
